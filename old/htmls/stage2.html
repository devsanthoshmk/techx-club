Basic Programming - Python

  1. What will print(bool([])) output in Python?
    1. True
    2. False
    3. Error
    4. None
      Answer: b) False
  2. Which of the following is used to handle exceptions in Python?
    1. try-except
    2. if-else
    3. switch-case
    4. error-handler
      Answer: a) try-except

Web Development

  1. Which tag is used to define an internal CSS in HTML?
    1. <css>
    2. <style>
    3. <link>
    4. <script>
      Answer: b) <style>
  2. What does ‘href’ in an anchor tag <a> define?
    1. Hyperlink Reference
    2. HTML Rendering Framework
    3. Header Reference
    4. Hypertext Result File
      Answer: a) Hyperlink Reference

Digital Marketing

    1. TV Advertisements
    2. Social Media Ads
    3. Radio Advertisements

      Answer: b) Social Media Ads

Data Structure and Algorithms

  1. Which of these data structures follows LIFO (Last In, First Out)?
    1. Queue
    2. Stack
    3. Array
    4. Linked List
      Answer: b) Stack
  2. (Hard - 5 marks) What is an advantage of using recursion in programming?
    1. It reduces function calls
    2. It simplifies complex problems
    3. It speeds up execution
    4. It decreases memory usage
      Answer: b) It simplifies complex problems

Database (MySQL)

  1. Which SQL statement is used to remove all records from a table but keeps its structure?
    1. DELETE
    2. REMOVE
    3. DROP
    4. TRUNCATE
      Answer: d) TRUNCATE
  2. Which database key is used to uniquely identify each row in a table?
    1. Foreign key
    2. Super key
    3. Candidate key
    4. Primary key
      Answer: d) Primary key

Operating Systems Concepts

  1. What is the main role of an operating system kernel?
  1. Manage files
  2. Manage hardware and software communication
  3. Display graphics
  4. Control network access
    Answer: b) Manage hardware and software communication
  1. (Hard - 5 marks) What is the primary difference between a process and a thread?
  1. A process is a lightweight operation, while a thread is heavyweight
  2. A thread is a lightweight operation, while a process is heavyweight
  3. Both are identical in execution
  4. Threads consume more memory than processes
    Answer: b) A thread is a lightweight operation, while a process is heavyweight

Basic Economics

  1. What is the term used for the total market value of all final goods and services produced within a country in a given year?
  1. Gross National Product (GNP)
  2. Net National Product (NNP)
  3. Gross Domestic Product (GDP)
  4. Consumer Price Index (CPI)
    Answer: c) Gross Domestic Product (GDP)
