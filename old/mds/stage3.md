### *Problem Solving (Python Coding)*  
#### *10. What will be the output of the following Python program?*  
```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(4))
```

a) 24  
b) 12  
c) 20  
d) 10  
*Answer:* a) 24
